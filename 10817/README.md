https://www.acmicpc.net/problem/10817

```
$ echo 20 30 10 | go run main.go
20
```

---

## 문제

세 정수 A, B, C가 주어진다. 이때, 두 번째로 큰 정수를 출력하는 프로그램을
작성하시오.


## 입력

첫째 줄에 세 정수 A, B, C가 공백으로 구분되어 주어진다. (1 ≤ A, B, C ≤ 100)

## 출력

두 번째로 큰 정수를 출력한다.

## 예제

```
20 30 10 -> 20
```

```
20 30 10 -> 30
```

```
40 40 40 -> 40
```

```
20 10 10 -> 10
```
